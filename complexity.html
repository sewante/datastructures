Complexity of an Algorithm

This refers to the measure of how fast an algorithm is and what amount of memory it uses, also known as time and space complexities. respectively.
The execution time and memory are the basic resources during the computation of an algorithm.

Running time Of an algorithm

This is the time taken by an algorithm to execute.Its also known as the execution time of an algorithm. The running time of an algorithm depends on mainly two factors, that is to say the input size data and also the software and hardware environment.

There are many algorithms in different programming languages with different syntax which can be written to solve the same problem. Therefore in real sense, an algorithm that takes less execution time and less memory would be more efficient to be used for solving a problem.
